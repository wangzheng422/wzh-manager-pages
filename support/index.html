wzh manager Support

wzh manager helps users manage EC2 instances in the AWS us-west-2 region from iPhone.

Getting Help

For support, contact the developer through the public GitHub profile at github.com/wangzheng422.

Important Notes

The app does not provide AWS credentials. Users import their own AWS environment values and are responsible for their AWS account permissions and EC2 usage costs.

AWS credential profiles are stored locally inside the app container and are not displayed on the home screen.
