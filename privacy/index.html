wzh manager Privacy Policy

Effective date: July 9, 2026

wzh manager is an iOS utility for managing EC2 instances in a user-owned AWS account.

Data Collection

The developer does not collect data from this app.

AWS Credentials

Users may import their own AWS environment values into the app. These values are stored locally inside the app container on the user's device and are used to sign AWS API requests initiated by the user or by user-configured iOS Shortcuts automation.

The app does not include bundled AWS credentials and does not send AWS credentials to the developer.

Network Requests

The app communicates with AWS services to list and manage EC2 instances in the user's AWS account. These requests are made directly between the user's device and AWS.

Third-Party Analytics and Advertising

The app does not use third-party analytics SDKs or advertising SDKs.

Contact

For privacy questions, contact the developer through the public GitHub profile at github.com/wangzheng422.
